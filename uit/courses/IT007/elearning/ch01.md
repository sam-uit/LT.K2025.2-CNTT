#### Cấu Trúc Lưu Trữ

Phân cấp (hierarchy):

- Tốc độ truy xuất (speed)
- Chi phí (cost)
- Bất biến khi không có điện (Volatility).

Volatile Storage (Primary):

- Registers
- Cache
- Memory

Non-volatile Storage (Secondary):

- Non-volatile memory
- HDD

Tertiary:

- Optical Disk
- Magnetic Tapes

#### Cấu Trúc Nhập Xuất

Các cách:

1. Cách 1: Write-through. Sử dụng lệnh ngắt.
2. Cách 2: Write-back.
